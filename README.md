# Calculadora Moderna - Python

Versión de la calculadora reimplementada en **Python** usando **tkinter** (librería estándar).

## Características

✅ Interfaz idéntica a la versión JavaScript  
✅ Operaciones matemáticas básicas (+, −, ×, ÷)  
✅ Funciones adicionales (%, ±, AC)  
✅ Diseño moderno con tema oscuro  
✅ Respuestas a errores (división por cero)  

## Requisitos

- Python 3.7+
- tkinter (incluido por defecto en la mayoría de instalaciones de Python)

## Cómo ejecutar

```bash
python calculator.py
```

O si tienes Python 3 instalado como `python3`:

```bash
python3 calculator.py
```

## Archivos

- **calculator.py** - La aplicación completa de la calculadora
- **index.html** - Interfaz web original (JavaScript)
- **app.js** - Lógica original en JavaScript
- **styles.css** - Estilos CSS originales

## Diferencias con la versión JavaScript

La lógica de cálculo es idéntica, pero la interfaz se construye con tkinter en lugar de HTML/CSS. El resultado visual es muy similar al original.
